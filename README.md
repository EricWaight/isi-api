# isi-api

## Project Dependencies: 
### -Docker desktop
### -Node/NPM

## Instructions to run:
### From the terminal:
### -Run docker-compose up -d in the main folder
### -Run 'npm install'
### -Run 'npm run seed'
### -Run 'npm run devStart'
